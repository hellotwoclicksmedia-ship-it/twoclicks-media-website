# Design Guidelines: TwoClicks Media

## Design Philosophy
**Brand Personality:** Notion × Linear × calm CRO studio energy
- Bold, Minimal, Calm, Premium, Conversion-first
- No playful colors, no loud gradients, no "agency orange"
- Authority through restraint, trust through clarity

## Color System

### Primary Palette
- **Primary Black:** #0B0D10 (backgrounds, headers, hero sections)
- **Soft White:** #F7F8FA (light backgrounds, content areas)

### Secondary Neutrals
- **Slate Gray:** #2A2F36 (section backgrounds, cards, dividers)
- **Muted Gray:** #9AA0A6 (microcopy, captions, metadata)

### Accent Colors
- **Conversion Green:** #3EE98A (PRIMARY ACCENT - CTAs, active states, hover highlights, key metrics)
  - **Critical Rule:** Use <8% of page surface. Scarcity = premium feel
- **Electric Blue:** #6EA8FF (RARE - links, focus states, charts only)

## Typography

### Primary Font: Inter
- **Headlines:** Weight 500-700, slightly tight tracking, sentence case (never ALL CAPS)
- **Body:** Weight 400-500, line-height 1.6-1.75
- **Usage:** All text, maintaining calm editorial feel

## Layout & Spacing

### Container Strategy
- **Full-width sections:** w-full with inner max-w-7xl
- **Content sections:** max-w-6xl  
- **Text content:** max-w-prose

### Vertical Rhythm
- Desktop section padding: py-20 to py-32
- Mobile section padding: py-12
- Consistent breathing room between all sections

### Responsive Breakpoints
- Mobile (base): Single column, stack everything
- Tablet (md:): Maximum 2 columns
- Desktop (lg:): 2-4 columns for feature grids, testimonials, stats

## Component Design

### Hero Sections
- Calm animation: subtle parallax or text reveal on scroll
- No loud visuals - focus on typography and generous spacing
- Include hero image for visual impact (premium product/workspace photography)
- Primary + secondary CTAs with micro-trust line underneath

### CTAs
- **Primary:** Solid button with Conversion Green background
- **Secondary:** Ghost/outline button or text link
- **Sticky header CTA:** "Get a Shopify Conversion Audit" always visible

### Cards
- Slate Gray (#2A2F36) backgrounds
- Generous padding (p-8 to p-12)
- Subtle hover states (slight elevation, no dramatic transforms)

### Forms
- Clean, minimal input fields
- Clear labels above inputs
- Conversion Green focus states
- Progressive disclosure for multi-step forms

## Animation Guidelines
- **Use sparingly:** Calm, purposeful motion only
- **Hero:** Subtle parallax or fade-in text reveals
- **Scroll:** Gentle fade-up on section entry
- **Hover:** Subtle scale (1.02x) or slight elevation
- **No:** Loud transitions, spinning elements, aggressive parallax

## Section-Specific Design

### Homepage Structure (8-10 sections)
1. Hero with headline, subhead, dual CTAs, micro-trust line
2. Problem/Empathy section (text-focused)
3. Proof strip with 3 key stats (horizontal layout)
4. Philosophy differentiation (2-column: text + supporting visual)
5. Services overview (4 cards in grid)
6. Process (4 steps, horizontal timeline or vertical)
7. Case study previews (2 featured cards)
8. Retainer positioning (centered, bold statement)
9. Ideal client qualification (bulleted list)
10. Final CTA (calm, confident)

### Navigation
- Sticky header with links: Services, Case Studies, Learning, Retainers
- Primary CTA button: "Audit" (Conversion Green)
- Clean, minimal design with generous spacing

### Footer
- Philosophy line at top
- Multi-column link grid (Services, Case Studies, Learning, etc.)
- Newsletter signup (optional, minimal design)
- Privacy/Terms links

## Images
- **Hero Section:** Yes - use premium workspace/product photography showing Shopify stores or DTC products
- **Case Studies:** Before/after screenshots, metric visualizations
- **Services:** Optional supporting imagery, prioritize clarity over decoration
- **About:** Team/workspace imagery optional, brand-first approach

## Accessibility
- Maintain WCAG AA contrast ratios
- Focus states visible (Electric Blue outline)
- Semantic HTML structure
- Keyboard navigation support

## Key Design Principles
1. **Breathing room over density:** Generous whitespace creates premium feel
2. **Hierarchy through typography:** Size, weight, and spacing - not decoration
3. **Conversion Green as strategic accent:** Guide eyes to actions
4. **Calm confidence:** No aggressive sales tactics, just clear value
5. **Editorial clarity:** Every word earns its place
